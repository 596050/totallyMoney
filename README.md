# Start steps:

- Enter the client's root directory

# 1. Provide .env files:

Change file name of .env-example to .env with

- REACT_APP_SERVICE_BASE_URL=/api
- PORT=3000

# 2. Run client:

- yarn && yarn start

# 3. To run tests:

For jest unit tests

- yarn test:unit

For cypress end to end tests

- yarn test:e2e
